# Sistem Informasi Toko Kelontong

## 1. Data Barang
- Create: Tambah barang baru; Inputnya:
Nama barang
Kategori (mie instan, minuman, rokok, sembako, dll.)
Harga beli
Harga jual
Stock
Satuan (pcs, bungkus, botol, dll.)
- Read: Tampilkan semua barang dalam tabel
- Update: Edit data barang (harga berubah? stock nambah?)
- Delete: Hapus barang yang sudah tidak dijual

## 2. Data Supplier
- Tambah supplier
- Tampilkan list supplier
- Edit data supplier
- Hapus supplier

## 3. Kategori Barang
- Tambah kategori (kategori mie, rokok, susu)
- List kategori
- Edit kategori
- Hapus kategori

## 4. Transaksi Pembelian Stok (ke supplier)
- Tambah pembelian (barang apa, jumlah berapa, dari siapa)
- List pembelian
- Edit pembelian
- Hapus pembelian

## 5. Penjualan (kasir sederhana)
- Tambah transaksi penjualan
- List transaksi
- Update transaksi
- Hapus transaksi

## Fitur Tambahan
- Upload foto barang (contoh: foto mie sedap, aqua, rokok dji sam soe)
- Validasi harga harus angka
- Validasi stok tidak boleh minus
- Filter kategori
- Pencarian barang
